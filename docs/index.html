Why jwx?

    
A modern JWT tool built for developers who care about simplicity and privacy.

    

      

        🎨
        
Beautiful Output

        
Colorized, syntax-highlighted token display in your terminal and browser. Timestamps are annotated with relative times.

      

      

        ⚡
        
Zero Dependencies

        
Single static binary. No runtime, no node_modules, no containers. Just download and run.

      

      

        🔒
        
Privacy First

        
Your tokens never leave your machine. Decoding happens entirely client-side in the browser and locally in the CLI.

      

      

        🌐
        
Open Source

        
MIT licensed and community driven. Inspect the code, contribute features, or fork it — it's yours.

      

    

  







  

    
Get Started

    
Install jwx with a single command.

    

      
Install via Go

      

        $ go install github.com/manimovassagh/jwx/cmd/jwx@latest
        
      

      
Requires Go 1.21+. Or download a prebuilt binary from Releases.
